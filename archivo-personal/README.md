# Archivo personal

Esta carpeta es para volcar contenido real fuera de la app.

Separa tu material por dominio para que luego podamos:

- importarlo a la web
- procesarlo con scripts
- migrarlo a JSON estructurado
- mantener una copia legible fuera del navegador

## Estructura

- `habitos/`: registros, tablas, notas o historiales de rutina
- `libros/`: listas, reseñas, lecturas terminadas o pendientes
- `series/`: seguimiento de series vistas o por ver
- `peliculas/`: historial de peliculas, notas y ratings
- `escritos/`: textos filosoficos, pensamientos, reflexiones
- `anecdotas/`: recuerdos, escenas y relatos personales
- `importaciones/`: CSV, exports o archivos crudos desde otras apps

## Formato sugerido

Puedes dejar:

- `.md` para texto humano
- `.json` si ya tienes estructura
- `.csv` para tablas o historiales

## Siguiente paso

Cuando empieces a llenar estas carpetas, el siguiente trabajo sera conectar este material con la app para importarlo o sincronizarlo.
